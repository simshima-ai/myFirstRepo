# マニュアル用 画面キャプチャ指示書

## 目的
マニュアル本文で使うスクリーンショットを、抜け漏れなく取得するためのチェックリストです。

## 取得ルール
- 解像度は固定（例: 1920x1080）
- 背景色/ズーム/用紙設定を統一
- 不要な選択状態は解除してから撮影
- キャプチャ名は `secXX_yyy.png` 形式
