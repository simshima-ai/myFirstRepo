S-CAD User Manual

Simple & Smart 2D CAD / v2.4.1

1. Overview

  • S-CAD is a browser-based 2D CAD tool.
  • No installation or registration is required.
  • You can save/load drawing data and export to PDF/SVG.

2. Interface

  • Center: drawing canvas
  • Left: create/edit tools
  • Right: snap, attributes, and settings panels

3. Quick Start

  1. Create a new drawing
  2. Draw geometry with the Line tool
  3. Add dimensions
  4. Add hatching as needed
  5. Save or export as PDF/SVG

4. Core Operations

  • Mouse wheel: zoom
  • Middle button or Alt + drag: pan
  • Click: select
  • Shift + click: multi-select
  • Delete: remove selection

5. Create Tools

  • Line / Rectangle / Circle / Arc
  • Position marker / Text / Dimension
  • Hatching / Double line

6. Edit Tools

  • Trim
  • Fillet
  • Vertex editing
  • Pattern copy

7. Layer Management

  • Add/delete/rename layers
  • Toggle visibility and lock state

8. Group Management

  • Group or ungroup multiple shapes
  • Select and move by group

9. Settings

  • Grid visibility and snap settings
  • Display quality, UI options, and helper overlays

10. Save / Load / Export

  • JSON save/load
  • SVG/PDF export

11. Troubleshooting

  • Cannot select objects: check layer lock state.
  • Slow rendering: reduce display quality and disable helper overlays.
