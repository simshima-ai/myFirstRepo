S-CAD User Manual

Simple & Smart 2D CAD / v2.4.1

1. Overview

  • S-CAD is a browser-based 2D CAD tool.
  • No installation or registration is required.
  • You can save/load drawing data and export to PDF/SVG.

2. Features

S-CAD is easy to operate, but understanding the concepts of objects, groups, and layers will make your workflow much smoother.

With a few exceptions, you can think of S-CAD geometry as being built from line segments and arcs. For example, a rectangle is a group made of four line-segment objects. Each corner of the rectangle actually has two vertices, but you can handle them as if they were a single vertex. You can also handle them separately when needed.

3. Quick Start

  1. Create a new drawing
  2. In Settings, choose unit, scale, and paper size
  3. Draw with tools such as Line, Circle, and Rectangle
  4. Save (JSON) or export to PDF/SVG/DXF

4. Core Operations

  • Mouse wheel: zoom
  • Middle button or Alt + drag: pan
  • Click: select
  • Shift + click: multi-select
  • Delete: remove selection

5. Selection Tool

The selection tool has Object Selection mode and Group Selection mode. You can switch between them with the Space key. Always keep in mind whether you are selecting objects or groups.

6. Edit Tools

7. Layers and Groups

Layers and groups are similar grouping mechanisms, and they make object management easier while increasing editing flexibility.

A group acts like a parent that bundles multiple objects. It can span across layers and can be moved or rotated. Groups can also have a hierarchy, allowing you to represent structural dependencies. This makes it possible to model articulated structures such as robot arms.

You can freely move objects and groups by dragging and dropping them in the Group panel. It is recommended to organize them periodically while working.

8. Trim and Fillet

S-CAD has a powerful Trim function. A practical approach is to draw intentionally intersecting lines first, then remove unnecessary parts with Trim. It is fine to adjust exact dimensions afterward. Of course, you can also draw everything to exact dimensions from the beginning.

Fillet creates a corner radius (R) between two touching objects.

There are two modes: "Trim" mode, which automatically removes unnecessary parts, and "No Trim" mode.

If Fillet removes a part you want to keep, create the corner radius in "No Trim" mode, then remove only the unwanted parts using the Trim tool.

You can change the radius of a created fillet later.

11. Attributes

There is a "Keep Attribute" button in the Snap panel.

When this is enabled, snapping a vertex in Vertex Edit mode can assign an attribute to that vertex, so it follows even if the snapped target object moves.

When an object with attributes is selected, the Attributes panel appears and you can remove attributes there.
