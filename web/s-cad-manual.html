01

概要

S-CAD はブラウザで動作する 2D CAD です。インストール不要・登録不要で、無料ですべての機能を利用できます。

ブラウザ動作 インストール不要 完全無料 JSON / PDF / SVG 入出力

以下の機能に対応しています。

  • 図形作成（線・四角・円・円弧・テキスト・寸法・ハッチング・二重線）
  • 図形編集（トリム・フィレット・頂点編集・パターンコピー）
  • レイヤー管理・グループ管理
  • 入出力（JSON 保存/読込・PDF/SVG 出力）
ℹ
左パネルに作成・編集・ファイル操作、右パネルにグループ・レイヤー管理を配置しています。

03

クイックスタート

初めて使う方は、以下の手順で基本的な図面作成の流れを確認できます。

  1. 新規作成 — ファイルパネルから「新規作成」を選択し、用紙サイズと縮尺を設定します。

  2. 外形を描く — 線ツールで外形を描きます。連続線モードを使うと頂点を連続して打てます。

  3. 整形する — フィレット（R付け）やトリム（不要部分削除）で形状を仕上げます。

  4. 寸法線を追加 — 寸法ツールで単一・直列・角度・円寸法を入れます。

  5. ハッチング — 閉領域の境界をクリックで選択し、ハッチングを作成します。

  6. レイヤー・グループ整理 — 要素をレイヤーやグループに整理して管理します。

  7. 保存・出力 — JSON 形式で保存、または PDF / SVG で出力します。

05

作成ツール

5.1 線
[ sec04_line_tool.png ] 線ツール（連続線モード含む）
  • 通常作成 — 2点を指定して線を引きます。
  • 連続線モード — クリックで頂点を追加し続け、Enter で確定します。
  • 長さ / 角度指定 — 数値入力で寸法を固定した線を作成できます。
  • サイズ固定 — 前回と同じ長さ・角度で連続作成します。
5.2 四角
[ sec05_rect_tool.png ] 四角ツール（基準点設定）
  • 幅 / 高さ指定 — 数値入力でサイズを固定して作成します。
  • 基準点（9点） — 左上・中央・右下など9箇所から基準点を選択できます。
  • サイズ固定 — 前回と同じサイズで連続作成します。
5.3 円
[ sec06_circle_modes.png ] 円ツール 3モード（ドラッグ / 半径固定 / 三点指示）
モード操作
マウスドラッグ中心点からドラッグして半径を決定
半径固定数値で半径を指定してクリック配置
三点指示円周上の 3 点をクリックして確定

中心マーカーの表示 / 非表示を切り替えられます。

5.4 円弧
[ sec07_arc_tool.png ] 円弧作成

開始角と終了角を持つ円弧を作成します。作成後は頂点編集ツールで角度を調整できます。

5.5 位置マーカー
[ sec08_position_text.png ] 位置マーカーとテキスト

基準点や補助目的のマーカーをキャンバス上に配置します。印刷・出力には含まれません（設定次第）。

5.6 テキスト

文字列をキャンバスに配置します。以下のプロパティを設定できます。

  • 文字列 — 入力テキスト
  • サイズ — フォントサイズ
  • 回転 — テキストの回転角
  • フォント — 使用するフォント種類
  • 色 — 文字色
5.7 寸法
[ sec09_dim_tools.png ] 寸法（単一 / 直列 / 角度 / 円）
種類用途
単一寸法2点間の距離を計測・表示
直列寸法複数の寸法を連続して配置
角度寸法2辺のなす角を表示
円寸法直径 / 半径を表示

矢印の形状・サイズ、文字サイズ、精度（小数点桁数）を設定できます。

5.8 ハッチング
[ sec10_hatch_single_pick.png ] ハッチ境界の単体選択
[ sec11_hatch_double_click_chain.png ] ハッチ境界のダブルクリック連続選択
  • 境界をクリック — 1本ずつ境界線を選択します。
  • 境界をダブルクリック — 接続する境界を自動的に連続選択します。
  • 空クリック — 境界の選択状態をすべて解除します。

ピッチ・角度・塗りつぶし・線色・塗り色を設定できます。

💡
ハッチング境界が選択できない場合は、対象図形が可視レイヤーにあるか、閉領域になっているかを確認してください。
5.9 二重線
[ sec12_doubleline.png ] 二重線生成（トリムあり / なし比較）

選択したオブジェクトから二重線（オフセット線）を生成します。

  • 線・円・円弧の組み合わせに対応
  • トリムあり — 交差部分を自動処理してきれいな二重線を生成
  • トリムなし — 元の形状をそのままオフセット
06

編集ツール

6.1 トリム
[ sec13_trim_fillet.png ] トリムとフィレット

交点付近の不要部分をクリックで削除します。複数の図形が交差している部分に使用してください。

6.2 フィレット

2つのオブジェクトを選択後、候補を確定してR（丸み）を付けます。

  • 半径 — フィレット半径を数値入力
  • トリムしない — 元の線を残してRを追加
  • トリムも行う — 元の線をRの接点でトリム
6.3 頂点編集
[ sec14_vertex_edit.png ] 頂点編集（スナップあり）

オブジェクトの頂点を個別に移動します。スナップと組み合わせることで精密な編集が可能です。

6.4 パターンコピー
[ sec15_pattern_copy.png ] パターンコピー（配列 / 回転 / 反転）
種類内容
配列X/Y 方向に指定数・間隔でコピー
回転基準点を中心に指定角度・数でコピー
反転指定軸に対して鏡像コピー

08

グループ管理

[ sec17_groups.png ] グループ操作（親子・複数選択・並び替え）
  • グループ作成 — 複数オブジェクトをまとめてグループ化
  • 親子構造 — グループをネストして階層管理が可能
  • 上下移動 — グループの並び順（表示順）を変更
  • リネーム — グループに名前を付けて識別しやすくする
  • カラー分け — グループごとに表示色を設定
  • 複数選択 — クリック / Shift+クリックで複数グループを同時選択

11

トラブルシューティング

オブジェクトが選択できない
[ sec21_troubleshooting_locked_layer.png ] LOCKレイヤー時の選択不可メッセージ
  • 対象オブジェクトのレイヤーが LOCK になっていないか確認してください。
  • 「選択レイヤーのみ編集」が有効になっていて、対象が非アクティブレイヤーにある可能性があります。
⚠
LOCKレイヤーのオブジェクトはグループ選択でもクリック選択でも選択できません。右パネルでロックを解除してから操作してください。
ハッチングの境界が選択できない
  • 対象の図形が 可視レイヤー（ON 状態）にあるか確認してください。
  • ハッチングには 閉領域 が必要です。線の端点がつながっているか確認してください。
動作が重い・描画が遅い
  • 設定で FPS 表示 を有効にして現在のフレームレートを確認します。
  • 不要なレイヤーを OFF にして描画対象を減らします。
  • 左メニューの 表示項目 を必要最小限に絞ります。
ℹ
設定の「オブジェクト数表示」を使って図面の規模を把握すると、パフォーマンス改善のヒントになります。
